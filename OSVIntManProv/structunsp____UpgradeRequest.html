enum unsp__NodeId* unsp__UpgradeRequest::NodeId

Indicates whether the request is for "Node 1" or Node 2".

Note: This operation is expected to be executed on Node 1 only; it is not supported for Node 2. To upgrade the second node, the unsp__UpgradeSecondNode() method must be used.
required: true
default value[range]: NA
