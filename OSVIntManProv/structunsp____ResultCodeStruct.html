int unsp__ResultCodeStruct::ResultCode

The Result Code of the request.

All error codes are eight digits long. The most significant digit (the first digit) is the category. The next two digits identify the object. The fourth and fifth digits are the sub-object. Digits six to eight represent the specific error. With this numbering scheme, we support 10 categories, 99 objects, 99 sub-objects for each object, and 999 unique return codes for each object/ sub-object. The following categories are defined:

  • Code: 1 (DB): Database errors.
  • Code: 2 (SYS): System errors (memory errors, file access)
  • Code: 3 (APP): An error detected in the client request or the request is invalid for the current state of the system objects.
  • Code: 9 (INT): Internal errors. In some cases, the objects being accessed may have been left in a corrupted state.
