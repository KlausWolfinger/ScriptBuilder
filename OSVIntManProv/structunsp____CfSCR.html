Detailed Description

Selective Call Rejecting.

The Selective Call Rejection feature provides the capability to build a list of numbers (known as a screening list) from which the subscriber does not want to accept incoming calls. When a caller's number does not match a number on the rejection list, the call is completed. When the caller's number matches a number on the rejection list, the caller hears an announcement that indicates the subscriber does not accept calls from the number.

MP2 command: CFSCR [Attributes]
