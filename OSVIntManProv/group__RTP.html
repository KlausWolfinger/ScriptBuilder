Detailed Description

RTP Parameters and Statistics Counters Definitions and Methods.

The RTP (Resilient Telco Platform) Management feature allows the convenient configuration of provisionable OpenScape Voice RTP parameters. Typically, RTP parameters are set during initial switch configuration to enforce global policies and ensure proper feature networking.

This method is used to retrieve and display the list of all RTP Statistics Counters.

The result data contains a list of RTP Statistics Counter names. For detailed information about one of the RTP Statistics Counters in the list, the GetStatCounterInfo operation can be used.
