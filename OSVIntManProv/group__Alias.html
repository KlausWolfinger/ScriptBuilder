Detailed Description

Alias Definitions and Methods.

Aliases can be associated with SIP endpoints. For SIP endpoints, up to five aliases are allowed. All Aliases created within OpenScape Voice must be unique. This is because these Aliases are used to determine where a received SIP request originated. An alias can be an IP address, an FQDN or the combination of IP address/FQDN and a port separated by a colon.
