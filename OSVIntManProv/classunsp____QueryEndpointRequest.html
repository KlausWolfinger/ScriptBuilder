xsd__int* unsp__QueryEndpointRequest::OutgoingCallSupervisionTimer

Allows filtering on the value of OutgoingCallSupervisionTimer.

The following rules apply:

  • >= 0 will retrieve all endpoints matching the timer
  • = -1 will retrieve all endpoints with timer != 0
  • < -1 will ignore timer filter.


required: false
default value[range]: -1 [0, 500-64000]

xsd__boolean unsp__QueryEndpointRequest::Specific

Indicates whether a specific (single) object is being requested, or a list of objects is being retrieved.

If True, any other input parameters are used to search for the exact object instance being requested. If False, any other input parameters are used as filter criteria and a list of objects matching the given parameters may be returned.
default value[range]: false
