Detailed Description

Alternative Calling Identity Delivery.

The Alternative Calling Identity Delivery feature lets administrator to be able to provision a list up to 10 alternative calling identities for an individual subscriber. The subscriber can dial a service access code and the 1 digit index into this list to select the calling party identity to be used prior to making an off-net call. It is not be possible to activate the Alternative CID feature without entries in the Alternative CID List.

MP2 command: CFACID [Attributes]
