wchar_t* unsp__AccountUserInfo::DisplayName

Used for in-switch and VPN dialing to present the calling party's name.

Used in installations where the SCP is unavailable or the switch is configured to use this field instead of the SCP. SIP phones may populate a generic name field that contains an alphabetical name to be associated with the number. If populated, that field will be used instead of this field. UTF-8 encoded characters are allowed.
required: false
default value[range]: NA [1..60]

char* unsp__AccountUserInfo::Pin

Specifies the subscriber's Personal Identification Number.

This code is available for use by various service features where a PIN has meaning, unless that feature has its own PIN. The subscriber PIN may be up to 10 characters long. Service feature pins may be up to 7 characters long.
required: false
default value[range]: NA [1..10]

char* unsp__AccountUserInfo::Pin2

Specifies an additional subscriber's Personal Identification Number.

All PINs (Pin-Pin5) are unique to a subscriber. PINs Pin2-Pin5 are available for use only during COSS service activation/deactivation. During COSS service activation and deactivation, the COSS service will validate the dialed PIN from all the PINs configured for that subscriber. Pin2-Pin5 may be up to 10 characters long. The valid characters for PINs are digits 0 to 9.
required: false
default value[range]: NA [1..10]

char* unsp__AccountUserInfo::PublicPin

A Public PIN provided by the subscriber that can be used by users who dial in.

For example, the ACR service requests a PIN before allowing the call to terminate to a subscriber. The Public PIN associated with the destination subscriber is expected to be input.
required: false

unsp__TimeZone unsp__AccountUserInfo::TimeZone

Specifies the subscriber's time zone location and determines how the subscriber's time-dependent services work.

In general, most subscribers will be in the switch's local time zone, but SIP phones can be anywhere and the switch could border a time zone edge and serve multiple time zones.
required: false
default value[range]: "Local"
