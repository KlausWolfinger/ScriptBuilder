Detailed Description

Call Forwarding Restrictions.

When assigned to a BGL, the call forwarding restrictions (CFR) feature restricts the type of destination for which this BGL can forward calls. Its arguments represent the types of calls that can be restricted as call forwarding destinations.

MP2 command: CFCFR [Attributes]
