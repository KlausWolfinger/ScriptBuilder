xsd__boolean* unsp__IPPhoneAttribute::AoRPerContact

Allows the support of single and multiple registration subscribers on the same switch.

  • If set to False, the switch will only allow one AoR per endpoint, either statically or dynamically registered.
  • If set to True, the switch will accept one AoR per DN and per each of the two aliases, store them in the contact and use them for terminating calls.


required: false
default value[range]: false
