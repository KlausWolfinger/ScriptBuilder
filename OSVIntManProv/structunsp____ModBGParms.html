unsp__NumberPlanName unsp__ModBGParms::BGCommonNumberPlanName

The name of a common numbering plan to be used when BGLs "escape" from a BG private numbering plan.

Only one BGCommonNumberPlanName is allowed per BG, Note that the specified numbering plan must be reserved for the BG before it can be assigned as the BG Common Numbering Plan.
required: false
