Detailed Description

Call Forwarding - All Calls.

The Call Forwarding - All Calls feature provides the capability to redirect calls intended for the subscriber to another destination. The subscriber activates and de-activates the feature and specifies the forwarding destination. Also known as Call Forwarding Variable or Call Forward Unconditional. This feature overrides CfCFB and CfCFDA.

MP2 command: CFCFV [Attributes]
