Detailed Description

File System Backup Definitions and Methods.

Openscape Voice provides methods to initiate and manage the backup of its file system.

Enumeration Type Documentation

enum unsp__FSRemoteProtocol

The protocol by which the backup file is to be transferred to the RemoteIPAddress.

Enumerator
FSB_sftp 

SFTP.

FSB_ftp 

FTP.

enum unsp__FSRetrievalMethod

Allows sorting by ServiceID-NID, EBGID, Monitors and Route Registrations.

Enumerator
FSB_pull 

Data will remain on the local host machine for retrieval.

FSB_push 

Data will be pushed to a remote host.
