unsp__AccessEZIPParm ( struct unsp__AccessEZIPParmRequest  AccessEZIPParmRequest,
struct unsp__AccessEZIPParmResult *  AccessEZIPParmResult 
)

This method is used to Set, Modify, Delete or Get Easy IP (EZ-IP) parameters.

Easy IP (EZ-IP) is a component responsible for changing configuration of the cluster node(s). It is used to update current configuration or restore previous system configurations(s) and is delivered in windows through NCPE package. It is also invoked by the image installation software to update configuration of the reference system during the image installation procedure and delivered in the Linux through UNSPifw package. There is no specific definition for each parameter, only a string that acts as the container of a parameter and its defined value. Note: All actions that include keyword Set, Mod or Del, initiate a job. Any action with Get keyword returns the list of parameters in the result, without running in the background.

unsp__ContactMe ( struct unsp__ContactMeRequest  ContactMeRequest,
struct unsp__ContactMeResult *  ContactMeResult 
)

The ContactMe mechanism allows the applications like UC Server and DLS to notify the subscriber endpoints (OSMO Mobile, OSMO Tablet, NG Phone) with server updates and (optionally) ask for these applications to contact the server on receiving the event.

The notification mechanism shall be generic and few applications for this mechanism are:

  • Presence Notifications
  • New IMs
  • DLS Server Updates (new configuration data, software upgrades etc)
  • New Voicemails
  • Web Collaboration Events

This method displays licensing information.

  • Licensing Information on the OpenScape Voice system for Dynamic Session licenses. It controls the number of SIP subscribers that are concurrently registered in the system.
  • Licensing Information on the OpenScape Voice system for OSMO licenses. It controls the number of mobile devices subscribed to the system.
  • Licensing Information on the OpenScape Voice system for SUSE SLES.
