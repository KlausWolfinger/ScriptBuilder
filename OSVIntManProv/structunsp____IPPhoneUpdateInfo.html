enum unsp__IPConnRegTypes* unsp__IPPhoneUpdateInfo::RegType

Registration type.

  • For Static registration, the Signaling port and IP of the phone must be sent. IP phones connected through a Gateway may be pre-registered. Note that Static registration is not supported for Keyset subscribers.
  • The signaling Port may be sent. If it is not, it will default to 5060. If the IPConnectionType is set to TLS the port will default to 5061. The request will be rejected if the IP Address is sent. Note that if the registration type is changed from Static to Dynamic, the IP Address, if any was present, will be cleared, and the Port, if not sent, will be set back to the default (5060).
    required: false
    default value[range]: NA
