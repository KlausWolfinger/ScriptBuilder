xsd__time unsp__CACPrimaryLinkStatusData::LinkStatusDownTimeStamp

The time when the status for the link was changed to "LinkDown".

If the CACGroup has been created and no status notification update is provided, then this time defaults to an empty string. When the link status is changed to "LinkUp", this field is cleared.
