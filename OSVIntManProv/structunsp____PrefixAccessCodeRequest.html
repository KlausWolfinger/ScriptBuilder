int* unsp__PrefixAccessCodeRequest::dnMaxLen

The maximum length of the digits used for digit matching.

If dnMinLen is 0, dnMaxLen must be 0 also. dnMinLen and dnMaxLen must not overlap. Prefix Access Codes with with dnMinLen/dnMaxLen have priority over codes with the same digits without dnMinLen/dnMaxLen when checking for a match.
required: true
default value[range]: NA [1..30]
