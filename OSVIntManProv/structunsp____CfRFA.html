Detailed Description

Remote Feature Access.

Remote Feature Access was designed to activate/deactivate the SRS (see unsp__CfSRS) feature from a "remote" telephone (any DN other than the one to which the features are subscribed). RFA can only be used in conjunction with SRS; therefore, subscriber provisioning will block the assignment of RFA if SRS is not also subscribed or being subscribed in the same request.

MP2 command: CFRFA [Attributes]
