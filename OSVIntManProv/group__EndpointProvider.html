Detailed Description

Endpoint Provider Definitions and Methods.

The endpoint Provider expresses the telecom service provider for an NNI Endpoint. Once a Provider is created it can be associated with an NNI Endpoint only if the NNIType of the specific Endpoint is set to PSTNGateway.
