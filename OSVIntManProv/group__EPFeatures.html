Detailed Description

Endpoint Profile Features Definitions.

This Section describes the Features that can be assigned to Endpoint Profiles. In addition to the definitions on this page see also unsp__CfTRS. Features can be added to or removed from Endpoint Profiles with the unsp__UpdateEpFeatures2() method.
