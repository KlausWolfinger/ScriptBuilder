int* unsp__MlhgTermEntry::QueuePriority

Specifies the order in which this MLHG member shall search queues, assuming the terminal is a member of multiple MLHGs.

This value is stored on a per MLHG/per MLHG member basis, so a terminal has a separate value for each MLHG of which it is a member. Queues with lower number priorities are checked before higher number priority queues. Queues with the same priority may be checked in any order
default value[range]: NA [1..255]
