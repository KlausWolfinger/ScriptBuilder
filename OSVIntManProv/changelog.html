Change Log

Summary of Changes in Version 8.0.

V8.00.01.01

  • Added Call Forwarding Override feature
  • Mutual Authentication Endpoint Attribute for TLS/MTLS authentication
  • Version returned in Fit4More Format
  • EZIP Changes without Reboot
  • AlphanumericSipUri support for features.
  • AlphanumericSipUri field for subscriber.
  • Endpoint and Subscriber attributes.
  • Support for AlphanumericSipUri in translation APIs.
  • Added Domain Codes support.
  • Control Support of Alphanumeric SIP URI per CSTA Application
  • SOAP Subscriber/EP Profile Interface Cleanup

V8.00.01.02

  • Added ContactMe method
  • Removal of deprecated licensing types (EndUser, TrunkingSession and ClientAccess)

V8.00.01.03

  • CentralSBC field introduced for endpoints.
  • RegistViaCentralSBC field introduced for subscribers.
  • Validate Time Limited License Support
  • Added QueryResponsibleDomains method
  • SecureConnected and BG ACL per CSTA Application -Trusted field introduced for BGs.
  • UseBillingNumberForDP field introduced for endpoints.

V8.00.01.04

  • Add CFLoopExtended to non subscriber endpoints
  • BG option to send call rejected by a subscriber (603 Decline) to voicemail
  • Added FwdOnCallReject And FwdOnDND In Call Forwarding Voice Mail (CfVM)

V8.00.01.05

  • Removed residential FP and unused connection types
  • Removal of Endpoint Default Resource Management Type
  • Wrong certificate status.
  • BGDeptVariant field introduced for BG Departments.
  • EmergencyBGDeptName field introduced for BG Subnets.
  • ScanRate, ScanResponseTimeout, Passcode, ALIDBID fields removed from BG Subnets.
  • BGEmergencyDeptName field introduced for Subscribers.

V8.00.01.06

  • Removal of configurable MLHG Max Queue Size
  • Bulk Treatment Management

V8.00.02.01

  • Endpoint and subscriber SIP Timer Control and Audit Enhancements

V8.00.03.01

  • OSV trace configuration via MP2

V8.00.04.01

  • Allow unregistration of SBC Stale registrations

V8.00.05.01

  • Added VideoCallIsAllowed endpoint attribute
  • Added MTPEnabled endpoint attribute
