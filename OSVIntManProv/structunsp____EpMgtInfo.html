enum unsp__ImpactLevel* unsp__EpMgtInfo::ImpactLevel

This feature enables the administrator to configure the Impact Level of a subscriber.

The subscriber is informed when an incoming call originates from a lower security zone or when an outgoing call terminates in a lower security zone. Lower security zone includes all calls with Impact Level less than the one configured for the specific subscriber. When a user participates in a multiparty call such as conference, he/she is informed if only any of the participants in the conference has a lower security zone. The notifications is a short message display on the OpenStage device along with an audible indication e.g a 'beep' or a special ringback tone.
required: false
default value[range]: ImpactLevelUnclassified
