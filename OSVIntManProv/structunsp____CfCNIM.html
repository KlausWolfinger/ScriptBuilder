Detailed Description

CSTA Network Interface Device (NID) Service and CSTA Name Provider.

CNIM enables the support CSTA on OSV Network Interface Devices (NID). CNIM supports a subset of services and events that can be used by CSTA-enabled adjunct application to detect the presence of an inbound and outbound off-network call; lookup a name string based on the calling/called device number and associate the calling/called device name within the OSV call context via CSTA service.

MP2 command: CFCNIM [Attributes] (Endpoint Profile)
