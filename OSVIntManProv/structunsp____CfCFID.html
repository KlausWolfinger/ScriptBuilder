Detailed Description

Call Forward Invalid Destination.

The Call Forward Invalid Destination (=Call Diversion - Invalid Destination) service is used to divert incoming calls made to an invalid destination in the OSV to a preconfigured intercept destination, like an attendant console. The specified alternate target is any dialable DN (number) or Alphanumeric SIP URI and should successfully translate into valid destinations routable via the OpenScape Voice system. If the translation of the alternate target fails the call is routed to a Media Server Intercept. If no alternate target is specified, the service remains disabled, although assigned to the Endpoint Profile.

MP2 command: CFCFID [Attributes]
