Detailed Description

Certificate Definitions and Methods.

Certificate Management handles the use of Certificates and Certificate Revocation Lists (CRL) for Certificates, used on the SIP (MTLS) interface to initiate a TLS connection between AudioCodes M800-T1 and OpenScape Voice. Mutual authentication is performed for communication between servers. As a result, certificates from the PKI need to be deployed and exchanged when a communication channel is established. As part of this exchange, the certificates' validity needs to be verified and the certificate's revocation status needs to be checked with an up-to-date CRL.

OpenScape Voice provides the appropriate management interface for the administrator to preview and configure the settings of the Certificates and the Certificate Revocation Lists.
