Detailed Description

Endpoint Definitions and Methods.

Endpoints are external devices, e.g. Gateways, Proxies or SIP Trunks and also SIP endpoints, like SIP telephones and analog endpoints. An endpoint is a network component, such as an originating or terminating device. An endpoint can be a DN that does not have a number associated with it yet. An Endpoint Profile enables you to set parameters for that endpoint.

Endpoint Management Levels

Endpoints and Endpoint Profiles can be managed on the following two levels:

  • on system-wide level
  • on Business Group level
