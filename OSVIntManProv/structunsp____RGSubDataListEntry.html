char* unsp__RGSubDataListEntry::RateArea

The subscriber can belong to a specific Rate Area.

This Rate Area defines what constitutes a local call vs. a long distance call, as well as the routing for outbound calls. Outgoing calls are routed using the Rate Area. During routing, if a subscriber belongs to a Rate Area, those routes are given preference; otherwise, the system's default rate area (unassigned) is used. If the subscriber has no Rate Area assigned, the subscriber can access default routes only.
