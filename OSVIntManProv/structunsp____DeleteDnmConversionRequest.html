xsd__boolean* unsp__DeleteDnmConversionRequest::Transactional

Indicates whether processing of requested list of Definitions is to be treated as a transactional block where all entries are expected to succeed or fail as a whole.

When set to True, atomic operations will be followed by a Commit/Rollback statement allowing them to be treated as a single block. Potential failed atomic operations are identified and displayed as part of the result of this API when Transactional field set to False.
required: false
default value[range]: false
