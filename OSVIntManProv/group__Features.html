Detailed Description

Subscriber Features Definitions.

This Section describes the Features that can be assigned to Subscriber and Feature Profiles.

  • For subscribers, features can be added, modified or removed with the unsp__UpdateSubscriberFeatures2() method.
  • For Feature Profiles, features can be added, modified or removed with the unsp__UpdateFeatureProfile2() method.
