xsd__boolean* unsp__GetNumberPlanListRequest::BGCommon

Indicates whether the list of BG Common Number Plans is to be retrieved.

If the field is not provided, there shall be no filtering on BG Common Numbering Plan. If the field is provided, and the value is false, only Numbering Plans that are not BG Common Numbering Plans will be returned. If the field is provided, and the value is true, only Numbering Plans that are BG Common Numbering Plans will be returned
required: false
default value[range]: false

xsd__boolean* unsp__GetNumberPlanListRequest::CombinedList

Indicates whether results should include global number plans and BG specific number plans.

  • If a BG Name is specified, CombinedList indicates whether the returned list should include system wide Numbering Plans as well as those associated with the specified BG.
  • If no BG Name is specified, CombinedList indicates whether the returned list should include all BG-related Numbering Plans as well as the system wide Numbering Plans
